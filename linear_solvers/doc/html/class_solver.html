Detailed Description

template<class T>
class Solver< T >

Implementation of solvers (class)

This class defines five solvers which calculate solutions array for linear systems entered from interface. The solver methods take Matrix<T> (or CSRMatrix<T>), b_right_value and b_right_size as most important arguments.

TThe type of data stored in the table
