\item Users may select to generate a specified size matrix with random number between wanted minimum and maximum value or read data from a {\ttfamily .txt} file. \item The default input data file is stored in {\ttfamily linear-\/solver\textbackslash{}acse-\/5-\/assignment-\/acse-\/5-\/team-\/for-\/juejue\textbackslash{}linear\+\_\+solvers\textbackslash{}data} as \href{linear_solvers/data/input.txt}{\texttt{ input.\+txt}}. \begin{DoxyItemize}
\item Users can select their own data file as data source but the data format {\bfseries{M\+U\+ST BE T\+HE S\+A\+ME AS A\+B\+O\+VE}}. You may find an example file called \href{linear_solvers/data/input.txt}{\texttt{ input.\+txt}} in {\ttfamily \textbackslash{}data} folder
\item Users can select four linear solvers to solve the linear system\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Iterative Jacobi Method -- Users need to input iteration times, initial guess for all solutions and wanted precision
\item Iterative Gauss-\/\+Seidel Method -- Users need to input iteration times, initial guess for all solutions and wanted precision
\item Direct Gaussian Elimination Method -- No need for extra input
\item Direct LU Decomposition -- No need for extra input
\item Iterative S\+OR Method -- Users need to input iteration times, initial guess for all solutions and wanted precision
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Users can select three output\+:
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Print all solutions as x vector with wanted precision and solver running time. Note that if no solution or the solution cannot meet precision requirement, the system will not output the result, but users can still choose to output results to a {\ttfamily .txt} file (default \href{linear_solvers/data/output.txt}{\texttt{ output.\+txt}} in {\ttfamily \textbackslash{}data} folder). Moreover, the system may show 0 ms sometimes. This is due to small size of the matrix (usually smaller than 50 x 50).
\item Output all data to a txt file. Users can save the output data file (default in {\ttfamily \textbackslash{}data} folder).
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item An example of format of {\ttfamily output.\+txt} is\+: (10 x 10 matrix)
\end{DoxyItemize}
